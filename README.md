## 🚀 Featured Projects & Deep Dives

<details>
<summary><b>🏗️ Project 1 — Enterprise Infrastructure & Pipeline Migration</b> &nbsp;|&nbsp; <code>AWS · Terraform · Jenkins · Prometheus</code></summary>

<br/>

> **Scope:** Architected the full migration of **30+ critical enterprise automation pipelines** across AWS regions, designed for maximum resilience, observability, and zero-downtime delivery.

| Dimension | Detail |
|---|---|
| **Infrastructure** | EC2/ECS environments provisioned end-to-end with Terraform & Terragrunt IaC modules |
| **Orchestration** | Jenkins HA (active-passive masters on NFS, F5 LB, custom HA manager scripts) |
| **Observability** | Prometheus + Telegraf + Grafana stack with live monitoring, dynamic service discovery, and alerting |
| **Scale** | Parallel triggering of 140+ build jobs with queue deduplication bypass strategy |
| **Reliability** | NFS validation, automatic state failover, JVM G1GC tuning (`-Xmx24g`) for high-throughput masters |

**🏆 Key Achievement:** Automated the complete cross-region migration strategy with **zero downtime**, integrating a real-time monitoring pipeline (`/proc/stat` CPU reads, disk I/O telemetry, pipe-delimited parsing) deployed as a live dashboard.

<br/>

</details>

| **Optimization** | Re-tuned Spark executor configs, instance fleet sizing, and S3 output partitioning strategies |

**🏆 Key Achievement:** Delivered a **compliance-clean, modernized big data platform** while measurably improving processing efficiency across petabyte-scale datasets — on schedule, with zero data loss.
